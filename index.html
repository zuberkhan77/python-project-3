Color Game

Welcome to the Color Game! This Python-based game challenges your focus and speed. The goal is simple: type the name of the color of the displayed word, not the text of the word itself. Test your reflexes and have fun!

How to Play:

  1. Run the Python script colorgame.py.
  2. Click inside the input field.
  3. Press Enter to start the game.
  4. Type the name of the color of the word displayed on the screen, not the text.
  5. Try to score as high as possible before the timer runs out!

This game uses Python's tkinter library for the graphical user interface. It also uses randomization to keep the game fresh and exciting.
